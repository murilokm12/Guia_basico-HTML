Guia Básico de HTML5

️Estrutura básica

📌 Todo documento HTML segue essa estrutura:


<!DOCTYPE html>
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8">
    <title>Minha Página</title>
  </head>
  <body>

    Criação aqui dentro
    
  </body>
</html>
<DOCTYPE html> → diz que o documento é HTML5 📜
<html> → raiz de toda a página 🌐
<head> → informações do site (título, links, meta) 🧠
<body> → tudo que aparece na tela 👀
<title> → nome que aparece na aba do navegador 🪪

<title>Google.com</title>

Saída:

<b> → Tabém torna o texto em negrito. 

  <p>

  <br>
  
  <b>frase em negrito com a tag b</b>

Links e 🌄 imagens

📌 <a href=" "> → cria links.
📌 <img src=" " alt=" "> → insere imagens. 


    
<a href="https://google.com">Ir para o Google</a>

<img src="imagem.png"  width="200">

Saída:

Ir para o Google

📋Listas

📌 <ul> → lista não ordenada (com marcadores).
📌 <ol> → lista ordenada (numerada).
📌 <li> → item da lista.

Exemplo de lista não ordenada


<ul>
  <li>Maçã</li>
  <li>Banana</li>
</ul>

Saída:

  • Maçã
  • Banana

Exemplo de lista ordenada


<ol>
  <li>Primeiro</li>
  <li>Segundo</li>
</ol>

Saída:

  1. Primeiro
  2. Segundo

Tabelas

📌 <table> → cria uma tabela.
📌 <tr> → linha da tabela.
📌 <th> → cabeçalho.
📌 <td> → célula comum. 


<table border="1">
  <tr>
    <th>Nome</th>
    <th>Idade</th>
  </tr>
  <tr>
    <td>Maria</td>
    <td>20</td>
  </tr>
</table>

Saída:

Nome Idade
Maria 20
